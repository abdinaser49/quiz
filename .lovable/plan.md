

# Generative AI & Higher Education Quiz

A single-page quiz application built with React, featuring questions about Generative AI and Higher Education in Somalia.

## Pages & Features

### 1. Welcome / Homepage
- Title: "Generative AI & Higher Education Quiz"
- Brief description of the quiz topic
- "Start Quiz" button
- Clean, modern design with an education-themed color scheme

### 2. Quiz Screen
- Displays one question at a time with 4 multiple-choice options
- Progress bar showing completion (e.g., "Question 3 of 10")
- Click an option to select it, then see immediate feedback:
  - ✅ Green highlight for correct answer
  - ❌ Red highlight for wrong answer (correct answer also shown)
- Score tracker visible during the quiz
- "Next Question" button appears after answering

### 3. Results Page
- Total score (e.g., "7 out of 10")
- Percentage with a visual indicator
- Performance message (e.g., "Excellent!", "Good job!", "Keep learning!")
- "Restart Quiz" button to try again

## Quiz Content
- 10+ sample questions covering Generative AI in Somali higher education, including topics like:
  - AI tools in academic settings
  - Ethical considerations
  - Impact on teaching and learning
  - Challenges and opportunities in Somalia's education system

## Design
- Fully responsive (mobile and desktop)
- Clean, minimal UI using the existing design system
- Smooth transitions between questions
- No backend needed — all data stored in-app
